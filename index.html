GPSA Shared CSS

Stylesheet CDN for Greater Peninsula Swimming Association

This subdomain serves shared stylesheets used across all GPSA websites and tools. Reference any file below using its absolute URL.

Available Stylesheets

  • gpsa-tools-common.css
    Shared styles for all GPSA web tools — headers, buttons, toasts, modals, brand colors
    https://css.gpsaswimming.org/gpsa-tools-common.css
  • gpsa-main.css
    Main site styles — imports roster, contacts, and officials stylesheets
    https://css.gpsaswimming.org/gpsa-main.css
  • gpsa-roster.css / gpsa-roster-contact.css / gpsa-roster-officials.css
    Roster table, contact table, and officials list styles for SwimTopia HTML exports
    https://css.gpsaswimming.org/gpsa-roster.css
  • invitationals-main.css
    Invitationals site styles — imports meet, table, store, and records stylesheets
    https://css.gpsaswimming.org/invitationals-main.css
